## Installation on MacOS
- Install Cmake from https://cmake.org/download
- Add Cmake to your path ```PATH="/Applications/CMake.app/Contents/bin":"$PATH"``` in your .zshrc or .bashrc
- ```cd lib/SEAL``` and type ```cmake .``` then ```make```
